# organismIDSource

## Definition 
URN denoting the authority from which the species identification is defined.

## Format
urn:catalog:[repository]:[institution]:[projectcode]:[OrganismID], e.g., urn:catalog:otn:Dalhousie:NSBS:Brandy-release
